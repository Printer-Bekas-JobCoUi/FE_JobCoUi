# bhl-admin-frontend (Frontend Demo)

Admin panel untuk:
**PLATFORM DIGITAL UNTUK MENINGKATKAN TRANSPARANSI, KEAMANAN, DAN AKSES KERJA BURUH HARIAN LEPAS DI INDONESIA**

✅ Fokus: *frontend-only* (sesuai tugas).  
✅ Ada halaman simulasi **Audit Blockchain** (dummy data) untuk menampilkan txHash/hash sebagai UI transparansi.

## Teknologi
- React + TypeScript
- Vite
- TailwindCSS
- React Router
- Lucide icons

## Menjalankan
```bash
npm install
npm run dev
```

Login: gunakan email apa saja (demo), klik **Masuk**.

## Struktur Halaman
- Dashboard
- Buruh
- Pemberi Kerja
- Pekerjaan
- Kontrak Digital
- Upah & Pembayaran
- Audit Blockchain
- Pelatihan
- Rating & Apresiasi
- Feed & Portofolio
- Jaminan Sosial
- Pengaturan
